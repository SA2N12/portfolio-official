BEWAY Intranet

BEWAY est une société de transport dans le secteur de la literie. L'entreprise dispose de plusieurs services ainsi qu'un service informatique. Le projet BEWAY Intranet a été développé pour améliorer la communication interne et la gestion des congés.

Contexte du projet

Conçu pour éliminer les processus internes fragmentés, Beway Intranet sertà centraliser les informations et faciliter la gestion des congés.

hub

Centralisation

Centralise et simplifie la diffusion d'informations internes à l'entreprise.

event_note

Gestion des congés

Centralise et simplifie les demandes de congé pour les salariés.

Particularités du projet

fingerprint

Authentification sécurisée

Protocoles de connexion exploitant un fichier Excel interne à l'entreprise pour s'assurer de l'authenticité des utilisateurs.

done_all

Confirmations de lecture

Un système de confirmation de lecture garantit que chaque annonce est effectivement consultée par les salariés.

notifications_active

Annonces en direct

Système de notifications avec envoi de mail et sms en temps réel pour s'assurer que les salariés soient informés.

Environnement technique

Frontend

React

Stylisé avec Tailwind CSS pour une flexibilité atomique.

Backend

Node.js

Framework Express pour un routage API évolutif.

Base de données

MongoDB

Stockage documentaire flexible pour des schémas de données variés.

Sécurité

Bcrypt

Hachage par sel conforme aux standards industriels pour la sécurité des identifiants.
